# Brain games

## How to install games

```
$ sudo npm install -g brain-games-zipofar
```

[![asciicast](https://asciinema.org/a/ImdoW8HHUwsI2jw0K9X4W39BV.svg)](https://asciinema.org/a/ImdoW8HHUwsI2jw0K9X4W39BV)

## Start game
Define number is even or not
```
$ brain-even
```
